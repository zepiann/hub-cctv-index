# hub-cctv-index

本仓库用于归档和发布多个独立的 HTML 页面。这些页面由不同来源生成或收集，以静态 HTML 文件的形式存放在仓库中，可通过 GitHub Pages 或其他静态托管服务进行访问。

## 项目简介

hub-cctv-index 是一个纯静态页面集合仓库，不依赖后端服务或数据库。每个 HTML 文件均为独立页面，可直接在浏览器中打开或通过 URL 访问。本仓库不针对任何特定域名、网站或平台，仅作为页面归档和索引的容器。

## 目录说明

```
hub-cctv-index/
├── index.html          # 主索引页面（可选）
├── pages/              # 归档的 HTML 页面存放目录
│   ├── example-1.html
│   ├── example-2.html
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：存放所有归档的 HTML 页面文件，可按功能或时间分类子目录。
- `assets/`：存放页面共用的样式、脚本、图片等静态资源。
- `index.html`：可选的根目录索引页，用于快速导航到各归档页面。

## 页面归档说明

- 每个 HTML 页面均为独立文件，不依赖于其他页面或外部 API。
- 页面内容可能来自不同来源或生成工具，本仓库仅做归档与发布。
- 归档页面可能不定期更新、新增或移除，以保持内容整洁。
- 所有页面均遵循通用 Web 标准，建议使用现代浏览器访问。

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/hub-cctv-index.git
   ```
2. 直接在浏览器中打开 `pages/` 目录下的 HTML 文件预览。
3. 若启用 GitHub Pages，可通过仓库设置指定源分支和目录，即可通过 `https://<username>.github.io/hub-cctv-index/` 访问。

## 维护说明

- 新增页面：在 `pages/` 目录下创建 HTML 文件，命名建议使用小写英文、数字和连字符。
- 更新页面：直接修改对应 HTML 文件内容，保持文件结构与现有规范一致。
- 删除页面：确认页面不再需要后，移出 `pages/` 目录并更新索引（如有）。
- 公共资源：如需添加全局样式或脚本，请放置在 `assets/` 对应子目录中，并在页面中引用。
